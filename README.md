# Evaluacion Parcial 3

Contexto:
La clínica SaludPlus utiliza un sistema informático para administrar pacientes, médicos,
citas médicas y pagos.
Actualmente todo el sistema funciona como una aplicación monolítica que centraliza
todos los procesos.
Con el aumento de pacientes, el sistema presenta:
• Lentitud al registrar citas médicas
• Problemas para integrarse con aplicaciones móviles
• Dificultad para mantener el sistema
La clínica desea migrar su sistema a una arquitectura de microservicios que permita
mayor escalabilidad e integración con sistemas externos.

Microservicios implementados:
Springboot
Java Script
Jupyter
Oracle SQL.
